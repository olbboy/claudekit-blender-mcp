# ClaudeKit Blender MCP - Brand Assets

Official logo and brand assets for ClaudeKit Blender MCP.

## Logo Files

| File | Description | Use Case |
|------|-------------|----------|
| `logo.svg` | Primary full-color logo | README, documentation, marketing |
| `logo-light.svg` | Dark text version | Light backgrounds, print |
| `logo-dark.svg` | White text version | Dark backgrounds, dark mode |
| `icon.svg` | Icon only (no text) | App icons, small displays |
| `favicon.svg` | Optimized favicon | Browser tabs, bookmarks |
| `logo-mono-black.svg` | Monochrome black | Print, fax, single-color |
| `logo-mono-white.svg` | Monochrome white | Dark print, embossing |

## Usage Guidelines

### Minimum Size
- Full logo: 120px width minimum
- Icon only: 32px minimum
- Favicon: 16px minimum

### Clear Space
Maintain clear space around the logo equal to the height of the "C" in ClaudeKit.

### Do's
- Use SVG format when possible for best quality
- Use `logo-light.svg` on light backgrounds
- Use `logo-dark.svg` on dark backgrounds
- Maintain aspect ratio when scaling

### Don'ts
- Don't stretch or distort the logo
- Don't change the colors
- Don't add effects (shadows, outlines, etc.)
- Don't place on busy backgrounds without contrast
- Don't rotate the logo

## PNG Export

Open `export-png.html` in a browser to generate PNG versions:
- Sizes: 16px to 2048px
- All exports have transparent backgrounds
- Select any logo variant before exporting
